# Паттерн: Command (Команда)

Создайте интерфейс `ICommand` с методом `Execute()`.
Реализуйте минимум две команды (например, `OpenCommand`, `SaveCommand`) с различным поведением.
Создайте класс-инициатор (invoker), который хранит ссылку на `ICommand` (поле, свойство или параметр конструктора) и вызывает `Execute()`.
