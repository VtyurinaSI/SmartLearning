# Паттерн: Decorator (Декоратор)

Создайте интерфейс `IComponent` с методом (например, `string Operation()` или `decimal GetCost()`).
Сделайте базовый компонент, реализующий интерфейс.
Создайте декоратор, который принимает `IComponent` в конструкторе, вызывает его метод и добавляет свое поведение.
При желании добавьте второй декоратор, чтобы показать цепочку оберток.
