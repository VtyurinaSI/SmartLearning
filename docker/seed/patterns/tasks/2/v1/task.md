# Паттерн: Singleton (Одиночка)

Создайте класс-одиночку (например, `Logger` или `Configuration`). Конструктор должен быть `internal`, чтобы ограничить создание экземпляров.
Предоставьте глобальную точку доступа через `public static` свойство `Instance` или метод `GetInstance`.
Добавьте простой метод (например, `Log(string message)`), чтобы показать использование экземпляра.
