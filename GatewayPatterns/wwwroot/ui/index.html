SmartLearning — UI (через Gateway)

JWT: нет UserId: нет API: /api

Настройки

Если UI хостится внутри Gateway, оставь поле пустым — будет использоваться /api. Если UI запущен отдельно — укажи базовый URL, например http://localhost:5000/api.

Регистрация

POST /api/auth/register → { token }

Логин

POST /api/auth/login → { token }

Прогресс пользователя

Использует Authorization и X-User-Id из JWT.

Проверка решения

Пинг через Gateway

UI подобран под твой Swagger: /api/auth/register, /api/auth/login, /api/progress/user_progress, /api/orc/check, /api/users/{msg}. Вариант без CORS — положи файлы в GatewayPatterns/wwwroot.
